# Documentation

This folder holds supporting documentation and reference waveforms.

## Adding a waveform screenshot

After running the simulation in QuestaSim:

1. In the Wave window, run `wave zoom full` to fit all transfers on screen.
2. Set radixes:
   - FSM state signals → right-click → Radix → Symbolic
   - Address / data buses → right-click → Radix → Unsigned
3. Capture the Wave window with **Win + Shift + S** (Windows) or your OS screenshot tool.
4. Save as `waveform.png` in this folder.
5. The image will then be referenced from the README.
